What we want
A grayscale image (8-bit) is a 2D matrix:
f ∈ {0, 1, . . . ,255}H×W
where each entry is a pixel intensity.

2 Step 2 — Build the Histogram (NumPy and Library)
2.1 Histogram Definition
For 8-bit grayscale, the histogram is:
h(k) = # {(x, y) | f (x, y) = k}, k = 0, . . . ,255
A normalized histogram is:
p(k) = h(k)
N , N = H · W

3 Step 3 — Histogram Equalization (HE)
3.1 Global HE Formula
Let p(k) be the normalized histogram and:
CDF(k) =
k∑
j=0
p(j)
The HE mapping is:
s(k) = ⌊255 · CDF(k)⌋
Then:
g(x, y) = s(f (x, y))
3.2 Implement HE using Pure NumPy
import numpy as np
# 1) Histogram + normalized
hist = np.bincount(img.ravel(), minlength=256)
p = hist / hist. sum()
# 2) CDF
cdf = np.cumsum(p) # length 256, in [0,1]
# 3) Mapping table (LUT)
lut = np.floor(255 * cdf).astype(np.uint8)
# 4) Apply LUT: new image g(x,y) = lut[f(x,y)]
img_he_np = lut[img]
Explanation
• We compute the CDF via np.cumsum.
• lut is a lookup table of size 256, mapping each old intensity to a new one.
• Applying HE becomes very fast: lut[img].
3.3 HE using a Library (OpenCV / scikit-image)
import cv2
# OpenCV global histogram equalization

from skimage import exposure

img_he_sk_u8 = (255.0 * img_he_sk).clip(0, 255).astype(np.uint8)

4.2 Automatic Thresholding: Otsu
Theory (Key Idea)
Otsu chooses T to maximize the between-class variance:
T ∗ = arg max
T
σ2
B(T )
It works best when the histogram is bimodal.
(A) NumPy Implementation (Otsu from Histogram)
import numpy as np
hist = np.bincount(img.ravel(), minlength=256).astype(np.float64)
p = hist / hist. sum()
omega = np.cumsum(p) # class probabilities for [0..T]
mu = np.cumsum(p * np.arange(256)) # cumulative mean
mu_t = mu[-1] # global mean
# Between-class variance for all thresholds
# sigma_b^2(T) = (mu_t * omega - mu)^2 / (omega*(1-omega))
# Avoid division by zero with a small epsilon or masking.
eps = 1e-12
sigma_b2 = (mu_t * omega - mu) ** 2 / (omega * (1.0 - omega) + eps)
T_otsu = int(np.argmax(sigma_b2))
img_otsu_np = np.where(img >= T_otsu, 255, 0).astype(np.uint8)
print("Otsu threshold (NumPy):", T_otsu)
Explanation
• We compute class probability ω0(T ) using cumulative sums.
• We compute cumulative means eﬀiciently (no loops over pixels).
• We evaluate σ2
B(T ) for all T and choose the maximum.
(B) Library Implementation (OpenCV / scikit-image)
import cv2
# If threshold=0 and we add THRESH_OTSU, OpenCV computes Otsu automatically
_, img_otsu_cv = cv2.threshold(img, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)
from skimage.filters import threshold_otsu

img_otsu_sk = (img >= T).astype(np.uint8) * 255

4.3 Linear Intensity Transform (Brightness/Contrast)
Theory
g = clip(af + b, 0, 255)
• a > 1: increase contrast; 0 < a < 1: decrease contrast
• b > 0: brighten; b < 0: darken
(A) NumPy Implementation
import numpy as np
a, b = 1.3, 15 # example parameters
img_lin_np = (a * img.astype(np.float32) + b)
img_lin_np = np.clip(img_lin_np, 0, 255).astype(np.uint8)
Explanation
• Convert to float before multiplication/addition to avoid overflow in uint8.
• Clip to the valid range and cast back to uint8.
(B) Library Implementation (OpenCV)
import cv2
a, b = 1.3, 15
img_lin_cv = cv2.convertScaleAbs(img, alpha=a, beta=b)
4.4 Gamma Correction
Theory
Gamma correction is typically applied on normalized intensities f ∈ [0, 1]:
g = f γ
• γ < 1: brightens dark regions
• γ > 1: darkens bright regions
(A) NumPy Implementation
import numpy as np
gamma = 0.7 # example
f = img.astype(np.float32) / 255.0 # normalize to [0,1]
g = np.power(f, gamma) # apply gamma
img_gamma_np = (255.0 * g).clip(0, 255).astype(np.uint8)
Explanation
• Always normalize to [0, 1] for stable gamma transformation.
• Convert back to uint8 for display and saving.
7
(B) Library Implementation (scikit-image)
from skimage import exposure
import numpy as np
gamma = 0.7
f = img.astype(np.float32) / 255.0
g = exposure.adjust_gamma(f, gamma=gamma) # returns float in [0,1]
img_gamma_sk = (255.0 * g).clip(0, 255).astype(np.uint8)
